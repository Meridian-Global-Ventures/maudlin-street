# Videos

<span class="archive-status" data-status="shell">shell — entries pending</span>

Video reference index: official promotional clips, live performance footage, documentary segments, and archived broadcasts. **No video is stored in this repo.** Records carry remote URLs only (John's external server, an authorized CDN, the Internet Archive, or rights-holder platforms).

<div class="archive-callout">
Video records carry: title, type (promo / live / documentary / interview-segment), source URL, archive URL where applicable, duration, rights notes, and the host platform. The repo enforces a forbidden-extension list (`.mp4`, `.mov`, `.webm`, etc.) via <code>scripts/check-repo.mjs</code> so media binaries cannot accidentally be committed.
</div>
