# Live

<span class="archive-status" data-status="shell">shell — entries pending</span>

A concert chronology for The Smiths and Morrissey, organised by tour and by year within each tour. PJLM's `live-moz.htm` and the Smiths live coverage are the primary authority for tour-level structure where they overlap.

<div class="archive-callout">
Live records carry: date, city, venue, country, tour name, supporting acts, setlist (when sourced), known recording status (audience / soundboard / circulating bootleg → cross-linked into <a href="{{ '/bootlegs/' | url }}">Bootlegs</a>), and source citations. Setlist data is treated as factual where multiple independent sources agree and as candidate-only when only a single secondary source is available.
