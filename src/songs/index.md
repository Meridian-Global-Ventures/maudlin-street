# Songs

<span class="archive-status" data-status="shell">shell — entries pending</span>

A flat alphabetical song index covering both The Smiths and Morrissey's solo catalogue, plus known cover versions either group performed. The Songs index is a navigational fan-out: each entry links forward into the relevant [Lyrics]({{ '/lyrics/' | url }}) note and [Releases]({{ '/releases/' | url }}) entry. Cover origins are noted in square brackets following the PJLM convention.

<div class="archive-callout">
Songs records carry: canonical title, performing artist (Smiths / Morrissey / both), writer credits, cover origin where applicable, first appearance (release id), and links to the corresponding lyric-note and release-record entries. Full lyric bodies and full performance audio are never reproduced.
</div>
