# Interviews

This section holds **interview metadata** - title, date, participants,
summary, rights notes, and references to external video URLs or
archival captures. The repo does not host video.

The schema for interview records lives in
`content/interviews/README.md`. Per-record fields include `title`,
`slug`, `date`, `participants`, `summary`, `transcript_status`,
`video_provider`, `video_url`, `poster_image`, `duration`,
`rights_notes`, and `source_ids`.

Video URLs point to remote hosts (John's external server, a future
CDN, or an archive). No video file is stored in this repo.

Real interview records will be authored in later phases. This is a
scaffold index page.
