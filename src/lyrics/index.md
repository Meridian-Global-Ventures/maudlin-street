# Lyrics

This section will hold lyric **notes** - short attributed annotations,
contextual commentary, and source linkage - **not** full lyric bodies.
Full lyrics from third-party rights holders are not republished here.

The schema for lyric content records lives in
`content/lyrics/README.md`. Per-record fields include `slug`,
`song_title`, `attributed_to`, `release_id` (when applicable),
`notes`, `quotes` (short attributed excerpts only), `source_ids`, and
`pjlm_authoritative_url` (the canonical local URL on the PJLM
reconstruction that the note draws from, if any).

Real lyric notes will be authored in later phases. This is a scaffold
index page.
